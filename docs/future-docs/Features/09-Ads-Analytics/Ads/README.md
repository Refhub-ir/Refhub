# مدیریت تبلیغات (Ads)

این زیرقابلیت به ادمین امکان مدیریت و نمایش تبلیغات در بخش‌های مختلف سایت را می‌دهد.

## امکانات:
- تعریف و مدیریت تبلیغات (بنر، متنی و ...)
- تعیین موقعیت نمایش تبلیغ (صفحه اصلی، جزئیات کتاب و ...)
- زمان‌بندی نمایش تبلیغات
- مشاهده آمار نمایش و کلیک هر تبلیغ

## نکات فنی:
- تبلیغات در دیتابیس ذخیره و از طریق پنل ادمین مدیریت می‌شوند
- امکان فعال/غیرفعال کردن تبلیغ بدون حذف آن
- بارگذاری تصاویر و لینک تبلیغ از طریق پنل ادمین
