# آمارگیری و تحلیل رفتار کاربران (Analytics)

این زیرقابلیت به ادمین امکان مشاهده و تحلیل آمار بازدید، کلیک و رفتار کاربران را می‌دهد.

## امکانات:
- ثبت و مشاهده آمار بازدید صفحات مختلف
- ثبت و مشاهده آمار کلیک روی تبلیغات
- مشاهده آمار خرید، ثبت‌نام و سایر رویدادهای مهم
- گزارش‌گیری و نمایش نمودارهای آماری

## نکات فنی:
- آمارها در دیتابیس یا سرویس‌های تحلیلی ذخیره می‌شوند
- امکان فیلتر و گروه‌بندی آمار بر اساس بازه زمانی، نوع رویداد و ...
- استفاده از ابزارهای تحلیلی (مانند Prometheus، Google Analytics و ...)
