# یادداشت‌گذاری و مدیریت یادداشت‌ها (Notes)

این زیرقابلیت به کاربران امکان ثبت، مشاهده و مدیریت یادداشت‌ها برای هر منبع را می‌دهد.

## امکانات:
- ثبت یادداشت توسط کاربر (در صورت لاگین بودن)
- مشاهده لیست یادداشت‌های ثبت‌شده برای هر منبع
- ویرایش و حذف یادداشت توسط نویسنده یا ادمین
- نمایش تعداد یادداشت‌های هر منبع

## نکات فنی:
- هر یادداشت به کاربر و منبع مرتبط است و در دیتابیس ذخیره می‌شود
- فقط نویسنده یادداشت یا ادمین می‌تواند آن را ویرایش یا حذف کند
- یادداشت‌ها به صورت real-time یا با رفرش صفحه نمایش داده می‌شوند
