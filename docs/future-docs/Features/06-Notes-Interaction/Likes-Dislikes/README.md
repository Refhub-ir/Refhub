# لایک و دیسلایک یادداشت‌ها (Likes & Dislikes)

این زیرقابلیت به کاربران امکان لایک یا دیسلایک یادداشت‌های ثبت‌شده را می‌دهد.

## امکانات:
- لایک و دیسلایک هر یادداشت توسط کاربران
- نمایش تعداد لایک و دیسلایک هر یادداشت
- جلوگیری از رأی تکراری توسط یک کاربر

## نکات فنی:
- هر رأی به کاربر و یادداشت مرتبط است و در دیتابیس ذخیره می‌شود
- رأی‌گیری به صورت atomic و با اعتبارسنجی کاربر انجام می‌شود
- امکان مشاهده لیست کاربران لایک‌کننده (در صورت نیاز ادمین)
