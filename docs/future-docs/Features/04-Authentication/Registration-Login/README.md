# ثبت‌نام و ورود (Registration & Login)

این زیرقابلیت به کاربران امکان ثبت‌نام و ورود به سامانه را می‌دهد.

## امکانات:
- ثبت‌نام با ایمیل و رمز عبور
- ورود با ایمیل/نام کاربری و رمز عبور
- ورود با گوگل و لینکدین (Social Login)
- بازیابی رمز عبور
- نمایش پیام‌های خطا و موفقیت

## نکات فنی:
- اعتبارسنجی اطلاعات ورودی در سمت کلاینت و سرور
- ارسال ایمیل فعال‌سازی پس از ثبت‌نام
- مدیریت سشن و کوکی‌ها پس از ورود
- پیاده‌سازی Social Login با OAuth2
