# احراز هویت چندمرحله‌ای (MFA)

این زیرقابلیت امنیت ورود کاربران را با اضافه کردن مرحله دوم (کد یکبار مصرف) افزایش می‌دهد.

## امکانات:
- فعال‌سازی MFA توسط کاربر (مثلاً Google Authenticator)
- ورود با وارد کردن کد یکبار مصرف پس از رمز عبور
- مدیریت و غیرفعال‌سازی MFA توسط کاربر
- بازیابی کدهای MFA (کدهای پشتیبان)

## نکات فنی:
- ذخیره وضعیت MFA برای هر کاربر در دیتابیس
- تولید و اعتبارسنجی کد یکبار مصرف (TOTP)
- امکان بازیابی یا ریست MFA در صورت فراموشی
