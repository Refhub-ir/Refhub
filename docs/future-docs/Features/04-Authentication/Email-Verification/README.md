# مدیریت و تأیید ایمیل (Email Verification)

این زیرقابلیت به کاربران امکان مدیریت ایمیل و تأیید صحت آن را می‌دهد.

## امکانات:
- ارسال ایمیل تأیید پس از ثبت‌نام یا تغییر ایمیل
- نمایش وضعیت تأیید ایمیل در پروفایل
- امکان ارسال مجدد ایمیل تأیید
- نمایش پیام موفقیت یا خطا

## نکات فنی:
- ارسال لینک تأیید به ایمیل کاربر
- اعتبارسنجی توکن تأیید هنگام کلیک روی لینک
- به‌روزرسانی وضعیت تأیید ایمیل در دیتابیس
