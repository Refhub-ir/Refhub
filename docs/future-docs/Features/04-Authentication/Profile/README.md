# مدیریت پروفایل کاربر (Profile)

این زیرقابلیت به کاربران امکان مشاهده و ویرایش اطلاعات پروفایل خود را می‌دهد.

## امکانات:
- مشاهده اطلاعات کاربری (نام، ایمیل، شماره تماس و ...)
- ویرایش اطلاعات پروفایل
- تغییر رمز عبور
- مشاهده تاریخچه فعالیت‌ها و سفارش‌ها

## نکات فنی:
- اطلاعات پروفایل در دیتابیس ذخیره می‌شود
- اعتبارسنجی اطلاعات جدید هنگام ویرایش
- امکان بارگذاری تصویر پروفایل (در صورت نیاز)
