# امکانات جانبی: اشتراک‌گذاری و تبلیغات (Sharing & Ads)

این زیرقابلیت امکانات جانبی صفحه کتاب را شامل می‌شود.

## امکانات:
- دکمه اشتراک‌گذاری لینک کتاب در شبکه‌های اجتماعی (تلگرام، واتساپ، توییتر و ...)
- نمایش تبلیغات مرتبط در کنار یا پایین صفحه کتاب
- نمایش منابع ویژه و پر بازدید در کنار صفحه

## نکات فنی:
- لینک اشتراک‌گذاری به صورت داینامیک و با اطلاعات کتاب ساخته می‌شود
- تبلیغات از طریق سیستم مدیریت تبلیغات پروژه بارگذاری می‌شود
- نمایش منابع ویژه با الگوریتم انتخاب خاص (مثلاً بر اساس بازدید یا امتیاز)
