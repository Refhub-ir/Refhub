# یادداشت و کامنت (Notes & Comments)

این زیرقابلیت به کاربران امکان ثبت، مشاهده و مدیریت یادداشت‌ها و کامنت‌ها برای هر کتاب را می‌دهد.

## امکانات:
- ثبت یادداشت یا کامنت توسط کاربر (در صورت لاگین بودن)
- مشاهده لیست یادداشت‌های ثبت‌شده برای هر کتاب
- لایک و دیسلایک یادداشت‌ها
- حذف یادداشت توسط نویسنده یا ادمین
- نمایش تعداد یادداشت‌های هر کتاب

## نکات فنی:
- هر یادداشت به کاربر و کتاب مرتبط است و در دیتابیس ذخیره می‌شود
- فقط نویسنده یادداشت یا ادمین می‌تواند آن را حذف کند
- لایک/دیس‌لایک به صورت atomic و با جلوگیری از رأی تکراری ذخیره می‌شود
- یادداشت‌ها به صورت real-time یا با رفرش صفحه نمایش داده می‌شوند
