# مدیریت کلیدواژه‌ها، دسته‌بندی‌ها، سری‌ها و زبان‌ها (Keywords, Categories, Series, Languages)

این زیرقابلیت به کاربران و ادمین امکان مشاهده و مدیریت ساختار منابع را می‌دهد.

## امکانات:
- نمایش لیست و جزئیات کلیدواژه‌ها (هر کلیدواژه لینک به صفحه منابع مرتبط دارد)
- نمایش لیست و جزئیات دسته‌بندی‌ها (هر دسته‌بندی لینک به صفحه منابع مرتبط دارد)
- نمایش لیست و جزئیات سری‌ها (هر سری لینک به صفحه منابع مرتبط دارد)
- نمایش لیست و جزئیات زبان‌ها (هر زبان لینک به صفحه منابع مرتبط دارد)
- امکان فیلتر منابع بر اساس هر مورد

## نکات فنی:
- هر مورد (کلیدواژه، دسته‌بندی، سری، زبان) به صورت لینک در صفحات منابع نمایش داده می‌شود.
- صفحات لیست و جزئیات برای هر مورد وجود دارد و قابل توسعه است.
- امکان افزودن، ویرایش و حذف توسط ادمین (در پنل مدیریت)
