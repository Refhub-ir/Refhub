# سیستم امتیازدهی، شمارنده بازدید و دانلود (Rating, View Count & Download)

این زیرقابلیت به کاربران امکان مشاهده و تعامل با امتیاز، تعداد بازدید و تعداد دانلود هر منبع را می‌دهد.

## امکانات:
- نمایش امتیاز میانگین هر کتاب (بر اساس رأی کاربران)
- امکان ثبت امتیاز توسط کاربر (در صورت لاگین بودن)
- نمایش تعداد بازدید هر کتاب (هر بار باز شدن صفحه، شمارنده افزایش می‌یابد)
- نمایش تعداد دانلود فایل کتاب (هر بار دانلود، شمارنده افزایش می‌یابد)

## نکات فنی:
- امتیازدهی معمولاً با ستاره یا اسلایدر انجام می‌شود و نتیجه به صورت میانگین نمایش داده می‌شود.
- شمارنده بازدید و دانلود به صورت atomic در دیتابیس ذخیره می‌شود تا دقت حفظ شود.
- فقط کاربران لاگین می‌توانند امتیاز ثبت کنند.
- امکان مشاهده لیست کاربران امتیازدهنده (در صورت نیاز ادمین)
