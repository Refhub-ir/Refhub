# دانلود فایل کتاب (Download)

این زیرقابلیت به کاربران امکان دانلود امن فایل کتاب (PDF یا سایر فرمت‌ها) را می‌دهد.

## امکانات:
- نمایش لینک دانلود برای کاربران مجاز (مثلاً بعد از خرید یا ثبت‌نام)
- شمارش تعداد دانلود هر کتاب
- نمایش پیام موفقیت یا خطا هنگام دانلود

## نکات فنی:
- لینک دانلود به صورت امن و با اعتبارسنجی کاربر ارائه می‌شود
- شمارنده دانلود به صورت atomic در دیتابیس ذخیره می‌شود
- امکان محدودیت تعداد دانلود برای هر کاربر (در صورت نیاز)
