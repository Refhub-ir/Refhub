# پرسش از هوش مصنوعی (AI Question)

این زیرقابلیت به کاربران امکان می‌دهد درباره کتاب یا محتوای آن از هوش مصنوعی سوال بپرسند و پاسخ دریافت کنند.

## امکانات:
- فرم ارسال سوال به هوش مصنوعی در صفحه کتاب
- نمایش پاسخ AI به صورت آنلاین و سریع
- امکان پرسش‌های متوالی و دریافت پاسخ‌های مختلف

## نکات فنی:
- ارتباط با سرویس AI (مثلاً OpenAI یا مدل داخلی)
- مدیریت صف و محدودیت تعداد پرسش برای هر کاربر
- ذخیره تاریخچه پرسش و پاسخ (در صورت نیاز)
