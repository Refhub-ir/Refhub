# کتاب‌های مرتبط (Related Books)

این زیرقابلیت لیستی از کتاب‌های مرتبط با کتاب فعلی را به کاربر نمایش می‌دهد.

## امکانات:
- نمایش کتاب‌های مرتبط بر اساس دسته‌بندی، سری، کلیدواژه یا نویسنده
- هر کتاب مرتبط به صورت کارت یا لیست با تصویر، عنوان و نویسنده نمایش داده می‌شود
- هر کتاب مرتبط لینک به صفحه جزئیات خود دارد

## نکات فنی:
- الگوریتم انتخاب کتاب‌های مرتبط قابل تنظیم است (مثلاً بر اساس بیشترین اشتراک کلیدواژه یا دسته‌بندی)
- نمایش کتاب‌های مرتبط در پایین یا کنار صفحه جزئیات کتاب
- امکان توسعه برای نمایش کتاب‌های پیشنهادی هوشمند
