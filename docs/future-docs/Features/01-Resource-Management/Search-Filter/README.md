# جستجو و فیلتر منابع (Search & Filter)

این زیرقابلیت به کاربران امکان می‌دهد منابع (کتاب، مقاله و ...) را بر اساس کلیدواژه، دسته‌بندی، نویسنده، زبان، سری و سایر فیلترها جستجو و محدود کنند.

## امکانات:
- جستجوی متنی (عنوان، توضیحات، نویسنده و ...)
- فیلتر بر اساس دسته‌بندی، کلیدواژه، زبان، سری، سطح مهارت و ...
- جستجوی سریع با موتور Meilisearch
- نمایش نتایج به صورت لیست یا شبکه‌ای
- امکان مرتب‌سازی نتایج (جدیدترین، پربازدیدترین، ...)

## ورودی‌ها:
- متن جستجو (Query)
- فیلترهای انتخابی (دسته‌بندی، زبان، ...)
- پارامترهای مرتب‌سازی

## خروجی:
- لیست منابع مطابق با جستجو و فیلتر
- تعداد کل نتایج
- اطلاعات خلاصه هر منبع (عنوان، تصویر، نویسنده، امتیاز، ...)

## نکات فنی:
- جستجو با Meilisearch انجام می‌شود و نتایج به صورت آنی نمایش داده می‌شود.
- فیلترها به صورت پارامترهای URL ارسال می‌شوند.
- امکان ترکیب چند فیلتر و جستجوی همزمان وجود دارد.
- نتایج جستجو قابل صفحه‌بندی (pagination) هستند.
