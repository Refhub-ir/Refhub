# ارسال منابع ویژه به تلگرام و لینکدین (Telegram & LinkedIn Integration)

این زیرقابلیت به ادمین امکان ارسال منابع ویژه یا پر بازدید به کانال تلگرام و صفحه لینکدین را می‌دهد.

## امکانات:
- ارسال خودکار یا دستی منابع ویژه به تلگرام
- ارسال خودکار یا دستی منابع ویژه به لینکدین
- مدیریت پیام و قالب ارسال شده

## نکات فنی:
- ارتباط با API تلگرام و لینکدین برای ارسال پیام
- مدیریت توکن و دسترسی‌های لازم برای ارسال
- امکان زمان‌بندی یا ارسال فوری منابع
