# پیشنهاد منابع مرتبط (Recommendation)

این زیرقابلیت به کاربران منابع مرتبط و پیشنهادی را بر اساس رفتار و علایق نمایش می‌دهد.

## امکانات:
- نمایش لیست منابع مرتبط با کتاب فعلی (بر اساس کلیدواژه، دسته‌بندی، نویسنده و ...)
- پیشنهاد منابع ویژه و پر بازدید
- نمایش منابع پیشنهادی در صفحه اصلی و صفحات جزئیات

## نکات فنی:
- الگوریتم انتخاب منابع مرتبط قابل تنظیم است (بر اساس اشتراک کلیدواژه، رفتار کاربر و ...)
- امکان توسعه برای پیشنهاد هوشمند با یادگیری ماشین
- نمایش منابع پیشنهادی به صورت کارت یا لیست با لینک به صفحه جزئیات
