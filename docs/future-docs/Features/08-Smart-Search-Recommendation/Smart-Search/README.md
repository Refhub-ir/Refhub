# جستجوی هوشمند منابع (Smart Search)

این زیرقابلیت به کاربران امکان جستجوی سریع و پیشرفته منابع را با استفاده از موتور جستجوی Meilisearch می‌دهد.

## امکانات:
- جستجوی متنی در عنوان، توضیحات، نویسنده و ...
- نمایش نتایج به صورت آنی و با سرعت بالا
- فیلتر نتایج بر اساس دسته‌بندی، کلیدواژه، زبان و ...
- مرتب‌سازی نتایج (جدیدترین، پربازدیدترین و ...)

## نکات فنی:
- استفاده از Meilisearch برای ایندکس و جستجوی منابع
- ارسال پارامترهای جستجو و فیلتر به صورت Query String
- نتایج جستجو قابل صفحه‌بندی (pagination) هستند
