# کمپین‌ها (Campaigns)

این زیرقابلیت امکانات مربوط به تعریف، نمایش و شرکت در کمپین‌های ویژه (تخفیف، رویداد و ...) را پوشش می‌دهد.

## امکانات:
- نمایش لیست کمپین‌های فعال و گذشته
- مشاهده صفحه جزئیات هر کمپین (توضیحات، تصویر، شرایط شرکت، وضعیت)
- امکان شرکت در کمپین (در صورت فعال بودن)
- نمایش کوپن یا تخفیف مرتبط با کمپین

## نکات فنی:
- هر کمپین دارای slug یکتا برای آدرس‌دهی است
- وضعیت کمپین (فعال/غیرفعال) و تاریخ شروع/پایان مدیریت می‌شود
- ارتباط کمپین با کوپن‌های تخفیف در دیتابیس
