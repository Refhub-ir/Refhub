# هدیه سازمانی (Organizational Gift)

این زیرقابلیت به سازمان‌ها و شرکت‌ها امکان ثبت درخواست هدیه گروهی یا چاپ اختصاصی کتاب را می‌دهد.

## امکانات:
- فرم ثبت درخواست هدیه سازمانی (نام، سازمان، نوع هدیه، تعداد، توضیحات و ...)
- مشاهده وضعیت درخواست (در انتظار، تماس گرفته شده، تکمیل شده، لغو شده)
- مدیریت درخواست‌ها توسط ادمین (بررسی، تغییر وضعیت، یادداشت ادمین)

## نکات فنی:
- هر درخواست هدیه سازمانی در دیتابیس ذخیره می‌شود
- وضعیت درخواست و یادداشت‌های ادمین قابل ویرایش است
- امکان ارسال پیام یا تماس با متقاضی توسط ادمین
