# مدیریت نویسندگان (Authors)

این زیرقابلیت امکانات مربوط به نمایش و مدیریت نویسندگان منابع را پوشش می‌دهد.

## امکانات:
- نمایش لیست نویسندگان با قابلیت جستجو و فیلتر
- مشاهده صفحه جزئیات هر نویسنده (بیوگرافی، تصویر، لیست کتاب‌ها)
- هر نویسنده لینک به صفحه جزئیات خود دارد
- نمایش تعداد کتاب‌های هر نویسنده

## نکات فنی:
- صفحه لیست نویسندگان قابل صفحه‌بندی و جستجو است
- صفحه جزئیات نویسنده شامل اطلاعات کامل و لیست کتاب‌های مرتبط است
- امکان افزودن، ویرایش و حذف نویسنده توسط ادمین (در پنل مدیریت)
