# مدیریت کارشناسان و انتخاب‌های پیشنهادی (Experts & Expert Picks)

این زیرقابلیت امکانات مربوط به نمایش و مدیریت کارشناسان و پیشنهادات آن‌ها را پوشش می‌دهد.

## امکانات:
- نمایش لیست کارشناسان با قابلیت جستجو و فیلتر
- مشاهده صفحه جزئیات هر کارشناس (تخصص، سوابق، خدمات)
- ثبت و ویرایش پیشنهاد (Expert Pick) توسط کارشناس
- مشاهده لیست انتخاب‌های کارشناسان (Expert Picks List)
- مشاهده جزئیات هر انتخاب کارشناس (Expert Pick)

## نکات فنی:
- صفحه لیست کارشناسان و انتخاب‌ها قابل صفحه‌بندی و جستجو است
- هر انتخاب کارشناس لینک به صفحه جزئیات خود دارد
- امکان افزودن، ویرایش و حذف کارشناس و انتخاب‌ها توسط ادمین (در پنل مدیریت)
