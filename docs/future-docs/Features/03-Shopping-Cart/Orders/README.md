# مدیریت سفارش‌ها (Orders)

این زیرقابلیت به کاربران امکان مشاهده و پیگیری سفارش‌های قبلی را می‌دهد.

## امکانات:
- مشاهده لیست سفارش‌های ثبت‌شده با وضعیت هر سفارش
- مشاهده جزئیات هر سفارش (اقلام، مبلغ، وضعیت پرداخت و ارسال)
- ثبت تجربه خرید برای هر سفارش
- مشاهده تاریخچه پرداخت‌ها و فاکتور

## نکات فنی:
- هر سفارش به کاربر و سبد خرید مرتبط است
- وضعیت سفارش (در انتظار پرداخت، پرداخت شده، ارسال شده و ...) قابل پیگیری است
- امکان ثبت تجربه خرید فقط برای سفارش‌های تکمیل‌شده
