# سبد خرید (Cart)

این زیرقابلیت به کاربران امکان مدیریت سبد خرید منابع را می‌دهد.

## امکانات:
- افزودن منابع (کتاب‌ها) به سبد خرید
- مشاهده لیست اقلام سبد خرید با تصویر، عنوان، قیمت و تعداد
- افزایش/کاهش تعداد هر آیتم
- حذف آیتم از سبد خرید
- مشاهده جمع کل مبلغ سبد خرید

## نکات فنی:
- سبد خرید برای هر کاربر به صورت جداگانه ذخیره می‌شود (در دیتابیس یا سشن)
- امکان مدیریت سبد خرید حتی برای کاربران مهمان (در صورت نیاز)
- به‌روزرسانی آنی سبد خرید با Ajax یا رفرش صفحه
