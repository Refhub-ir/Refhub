# پرداخت و نهایی‌سازی خرید (Checkout)

این زیرقابلیت به کاربران امکان ثبت سفارش و پرداخت آنلاین را می‌دهد.

## امکانات:
- مشاهده خلاصه سفارش و اطلاعات پرداخت
- انتخاب روش ارسال و وارد کردن اطلاعات ارسال
- اتصال به درگاه پرداخت آنلاین
- نمایش پیام موفقیت یا خطا پس از پرداخت

## نکات فنی:
- سفارش پس از پرداخت موفق ثبت و وضعیت آن به‌روزرسانی می‌شود
- اطلاعات پرداخت و سفارش در دیتابیس ذخیره می‌شود
- امکان ارسال ایمیل یا پیامک تایید سفارش
