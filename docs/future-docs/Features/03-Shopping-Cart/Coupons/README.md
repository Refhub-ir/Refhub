# مدیریت کوپن‌های تخفیف (Coupons)

این زیرقابلیت به کاربران امکان مشاهده و استفاده از کوپن‌های تخفیف را می‌دهد.

## امکانات:
- مشاهده لیست کوپن‌های فعال و قابل استفاده
- وارد کردن کد کوپن در سبد خرید یا پرداخت
- اعمال تخفیف روی مبلغ سفارش
- نمایش پیام موفقیت یا خطا هنگام استفاده از کوپن

## نکات فنی:
- اعتبارسنجی کوپن (تاریخ انقضا، تعداد استفاده، شرایط سفارش)
- ثبت استفاده از کوپن در دیتابیس
- امکان مدیریت کوپن‌ها توسط ادمین (افزودن، ویرایش، حذف)
