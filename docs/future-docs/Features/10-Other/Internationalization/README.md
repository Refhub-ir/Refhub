# سیستم چندزبانه (Internationalization/i18n)

این زیرقابلیت به کاربران امکان استفاده از سایت به زبان‌های مختلف را می‌دهد.

## امکانات:
- انتخاب زبان سایت توسط کاربر (فارسی، انگلیسی و ...)
- ترجمه رابط کاربری و محتوای صفحات
- ذخیره زبان انتخابی در سشن یا کوکی

## نکات فنی:
- استفاده از ابزارهای ترجمه Django (gettext، فایل‌های po/mo)
- مدیریت ترجمه‌ها در پنل ادمین یا فایل‌های پروژه
- امکان افزودن زبان جدید بدون تغییر کد اصلی
