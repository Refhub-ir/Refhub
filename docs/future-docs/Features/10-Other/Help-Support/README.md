# صفحات راهنما، قوانین و پشتیبانی (Help, Support, Rules)

این زیرقابلیت به کاربران امکان دسترسی به اطلاعات راهنما، قوانین سایت و ارتباط با پشتیبانی را می‌دهد.

## امکانات:
- نمایش صفحات راهنما و سوالات متداول (FAQ)
- نمایش قوانین و مقررات سایت
- فرم تماس با پشتیبانی یا ارسال تیکت
- نمایش پیام‌های موفقیت یا خطا پس از ارسال درخواست

## نکات فنی:
- صفحات راهنما و قوانین به صورت استاتیک یا داینامیک قابل مدیریت هستند
- پیام‌های پشتیبانی در دیتابیس ذخیره و توسط ادمین مدیریت می‌شوند
- امکان ارسال پاسخ به کاربر از طریق ایمیل یا پنل کاربری
