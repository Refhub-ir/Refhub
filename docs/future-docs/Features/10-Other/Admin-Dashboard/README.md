# داشبورد ادمین و مدیریت پیشرفته (Admin Dashboard)

این زیرقابلیت به ادمین امکان مدیریت کامل بخش‌های مختلف سایت را می‌دهد.

## امکانات:
- مدیریت کاربران، منابع، سفارش‌ها، یادداشت‌ها، تبلیغات و ...
- مشاهده و ویرایش اطلاعات هر بخش
- گزارش‌گیری و مشاهده آمار کلی سایت
- مدیریت سطوح دسترسی و نقش کاربران

## نکات فنی:
- استفاده از پنل مدیریت Django (Django Admin)
- امکان افزودن، ویرایش و حذف داده‌ها از طریق پنل
- قابلیت توسعه و سفارشی‌سازی پنل ادمین
