# کش صفحه خانه (Home)

- **نوع کش:** کش داده (Data Cache)
- **کلیدهای کش:**
  - home_data:{user.id} (داده‌های صفحه خانه برای هر کاربر)
  - home_user_data:{user.id} (داده‌های شخصی‌سازی شده برای هر کاربر)
- **مدت کش:** 30 دقیقه برای داده‌های کاربر، 1 ساعت برای داده‌های عمومی
- **پاک‌سازی:** پس از تغییر داده‌های مهم (مثلاً امتیاز کاربر، منابع ویژه)، کش مرتبط با کاربر یا داده عمومی پاک می‌شود.
- **هدف:** افزایش سرعت بارگذاری صفحه خانه و کاهش بار دیتابیس برای داده‌های پرمصرف و شخصی‌سازی شده.
