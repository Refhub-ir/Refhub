# کش لیست سری‌ها (refrence_series)

- **نوع کش:** کش صفحه (Page Cache) و کش داده
- **دکوریتور:** @cache_page(60 * 15) (مدت کش: 15 دقیقه)
- **کلید کش:** refrence_series_list
- **پاک‌سازی:** پس از افزودن/ویرایش/حذف سری، کش لیست سری‌ها پاک می‌شود.
- **هدف:** افزایش سرعت بارگذاری لیست سری‌ها و کاهش بار دیتابیس.
