# کش صفحه جزئیات منبع (refrence_detail)

- **نوع کش:** کش داده (Data Cache)
- **کلید کش:** refrence_detail_{slug}
- **مدت کش:** 15 دقیقه
- **موارد کش‌شده:** اطلاعات کامل منبع (کتاب)، تصویر، نقد و بررسی‌ها، سوالات مرتبط، تبلیغات مرتبط
- **پاک‌سازی:** پس از ثبت کامنت، امتیاز یا ویرایش منبع، کش مرتبط با آن slug پاک می‌شود (clear_refrence_cache)
- **هدف:** افزایش سرعت بارگذاری صفحه جزئیات و کاهش کوئری‌های سنگین به دیتابیس.
