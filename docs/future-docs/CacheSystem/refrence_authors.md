# کش صفحه لیست نویسندگان (refrence_authors)

- **نوع کش:** کش صفحه (Page Cache) و کش داده
- **دکوریتور:** @cache_page(60 * 15) (مدت کش: 15 دقیقه)
- **کلیدهای کش:**
  - authors_page_{page} (برای هر صفحه از لیست)
  - authors_total_count (تعداد کل نویسندگان)
- **پاک‌سازی:** پس از افزودن/ویرایش/حذف نویسنده، کش صفحات مرتبط پاک می‌شود.
- **هدف:** افزایش سرعت بارگذاری لیست نویسندگان و کاهش بار دیتابیس هنگام صفحه‌بندی.
