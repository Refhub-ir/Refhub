# کش لیست کلیدواژه‌ها (refrence_keyword)

- **نوع کش:** کش صفحه (Page Cache) و کش داده
- **دکوریتور:** @cache_page(60 * 15) (مدت کش: 15 دقیقه)
- **کلیدهای کش:**
  - keywords_page_{page} (برای هر صفحه از لیست)
  - keywords_total_count (تعداد کل کلیدواژه‌ها)
- **پاک‌سازی:** پس از افزودن/ویرایش/حذف کلیدواژه، کش صفحات مرتبط پاک می‌شود.
- **هدف:** افزایش سرعت بارگذاری لیست کلیدواژه‌ها و کاهش بار دیتابیس هنگام صفحه‌بندی.
