# کش صفحه لیست منابع (refrence_list)

- **نوع کش:** کش صفحه (Page Cache) و کش داده
- **دکوریتور:** @cache_page(60 * 15) (مدت کش: 15 دقیقه)
- **کلیدهای کش:**
  - refrence_list_page_{page} (برای هر صفحه از لیست)
  - refrence_total_count (تعداد کل منابع)
- **پاک‌سازی:** پس از افزودن/ویرایش/حذف منبع، کش صفحات مرتبط پاک می‌شود.
- **هدف:** افزایش سرعت بارگذاری لیست منابع و کاهش بار دیتابیس هنگام صفحه‌بندی.
