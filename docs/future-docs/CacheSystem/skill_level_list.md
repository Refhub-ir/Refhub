# کش لیست سطوح مهارت (skill_level_list)

- **نوع کش:** کش صفحه (Page Cache) و کش داده
- **دکوریتور:** @cache_page(60 * 15) (مدت کش: 15 دقیقه)
- **کلید کش:** skill_level_list
- **پاک‌سازی:** پس از افزودن/ویرایش/حذف سطح مهارت، کش لیست سطوح مهارت پاک می‌شود.
- **هدف:** افزایش سرعت بارگذاری لیست سطوح مهارت و کاهش بار دیتابیس.
