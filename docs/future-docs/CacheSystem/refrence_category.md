# کش لیست دسته‌بندی‌ها (refrence_category)

- **نوع کش:** کش صفحه (Page Cache) و کش داده
- **دکوریتور:** @cache_page(60 * 15) (مدت کش: 15 دقیقه)
- **کلیدهای کش:**
  - category_list (لیست دسته‌بندی‌ها)
  - category_children_{pk} (زیرمجموعه‌های هر دسته)
  - category_references_{pk}_page_{current_page} (منابع هر دسته در هر صفحه)
  - category_references_count_{pk} (تعداد منابع هر دسته)
- **پاک‌سازی:** پس از افزودن/ویرایش/حذف دسته‌بندی یا منبع، کش صفحات مرتبط پاک می‌شود.
- **هدف:** افزایش سرعت بارگذاری لیست دسته‌بندی‌ها و کاهش بار دیتابیس هنگام صفحه‌بندی.
