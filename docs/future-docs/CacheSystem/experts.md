# کش لیست کارشناسان (experts)

- **نوع کش:** کش صفحه (Page Cache) و کش داده
- **دکوریتور:** @cache_page(60 * 15) (مدت کش: 15 دقیقه)
- **کلید کش:** approved_experts
- **پاک‌سازی:** پس از تایید/حذف کارشناس، کش لیست کارشناسان پاک می‌شود.
- **هدف:** افزایش سرعت بارگذاری لیست کارشناسان و کاهش بار دیتابیس.
