# کش لیست زبان‌ها (language_list)

- **نوع کش:** کش صفحه (Page Cache) و کش داده
- **دکوریتور:** @cache_page(60 * 15) (مدت کش: 15 دقیقه)
- **کلید کش:** language_list
- **پاک‌سازی:** پس از افزودن/ویرایش/حذف زبان، کش لیست زبان‌ها پاک می‌شود.
- **هدف:** افزایش سرعت بارگذاری لیست زبان‌ها و کاهش بار دیتابیس.
