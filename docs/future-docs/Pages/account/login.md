# صفحه ورود (Login)

**مسیر احتمالی:** `/auth/login/` یا `/account/login/`

**کاربرد:**
این صفحه برای ورود کاربران به سامانه استفاده می‌شود. فرم ورود شامل فیلدهای نام کاربری/ایمیل و رمز عبور است. در صورت ورود موفق، کاربر به داشبورد یا صفحه اصلی هدایت می‌شود.

**امکانات ویژه:**
- امکان ورود با حساب گوگل و لینکدین (Social Login) نیز فراهم است و کاربران می‌توانند با استفاده از این سرویس‌ها وارد سامانه شوند.

**نکات مهاجرت به دات‌نت:**
- فرم ورود باید با سیستم احراز هویت دات‌نت یکپارچه شود.
- مدیریت پیام‌های خطا و ریدایرکت پس از ورود اهمیت دارد.
- پیاده‌سازی ورود با گوگل و لینکدین با استفاده از OAuth2 در دات‌نت انجام شود.
