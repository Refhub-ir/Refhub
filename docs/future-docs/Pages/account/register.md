# صفحه ثبت‌نام (Register)

**مسیر احتمالی:** `/auth/register/` یا `/account/signup/`

**کاربرد:**
این صفحه برای ثبت‌نام کاربران جدید استفاده می‌شود. فرم ثبت‌نام معمولاً شامل نام، ایمیل، رمز عبور و تأیید رمز عبور است.

**نکات مهاجرت به دات‌نت:**
- اعتبارسنجی سمت سرور و کلاینت.
- ارسال ایمیل فعال‌سازی پس از ثبت‌نام.
