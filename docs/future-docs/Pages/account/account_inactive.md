# صفحه غیرفعال بودن حساب (Account Inactive)

**مسیر احتمالی:** `/account/inactive/`

**کاربرد:**
این صفحه زمانی نمایش داده می‌شود که حساب کاربر غیرفعال باشد (مثلاً پس از ثبت‌نام تا تأیید ایمیل یا توسط ادمین).

**نکات مهاجرت به دات‌نت:**
- نمایش پیام مناسب به کاربر.
- امکان درخواست فعال‌سازی مجدد یا راهنمایی برای رفع مشکل.
