# صفحه خطای پرداخت (Payment Error)

**مسیر احتمالی:** `/financial/payment/error/`

**کاربرد:**
این صفحه زمانی نمایش داده می‌شود که پرداخت کاربر با خطا مواجه شود و پیام خطا و راهنمایی لازم را ارائه می‌دهد.

**نکات مهاجرت به دات‌نت:**
- نمایش پیام خطا و راهنمایی برای رفع مشکل.
- امکان تلاش مجدد یا بازگشت به سبد خرید.
