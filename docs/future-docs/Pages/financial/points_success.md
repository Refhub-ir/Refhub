# صفحه موفقیت خرید امتیاز (Points Success)

**مسیر احتمالی:** `/financial/points/success/`

**کاربرد:**
این صفحه پس از خرید موفق امتیاز به کاربر نمایش داده می‌شود و پیام موفقیت‌آمیز بودن تراکنش را نشان می‌دهد.

**نکات مهاجرت به دات‌نت:**
- نمایش پیام موفقیت و اطلاعات تراکنش.
- ریدایرکت مناسب به صفحه امتیازات یا داشبورد.
