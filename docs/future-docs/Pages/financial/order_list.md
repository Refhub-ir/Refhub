# صفحه لیست سفارش‌ها (Order List)

**مسیر احتمالی:** `/financial/orders/`

**کاربرد:**
این صفحه لیست سفارش‌های قبلی کاربر را نمایش می‌دهد و امکان مشاهده جزئیات هر سفارش را فراهم می‌کند.

**امکانات ویژه:**
- برای هر سفارش، دکمه‌ای جهت ثبت تجربه خرید (نظر یا امتیازدهی) وجود دارد تا کاربر بتواند مستقیماً از همین لیست تجربه خود را ثبت کند.

**نکات مهاجرت به دات‌نت:**
- نمایش اطلاعات سفارش‌ها با فیلتر و مرتب‌سازی.
- نمایش وضعیت هر سفارش.
- پیاده‌سازی دکمه ثبت تجربه خرید برای هر سفارش و اتصال آن به فرم مربوطه.
