# صفحه سبد خرید (Cart)

**مسیر احتمالی:** `/financial/cart/`

**کاربرد:**
این صفحه اقلام انتخاب‌شده توسط کاربر برای خرید را نمایش می‌دهد. کاربر می‌تواند اقلام را حذف یا تعداد آن‌ها را تغییر دهد و به مرحله پرداخت برود.

**نکات مهاجرت به دات‌نت:**
- مدیریت سبد خرید در سمت سرور یا کلاینت.
- هماهنگی با سیستم پرداخت جدید.
