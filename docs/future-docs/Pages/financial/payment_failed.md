# صفحه شکست پرداخت (Payment Failed)

**مسیر احتمالی:** `/financial/payment/failed/`

**کاربرد:**
این صفحه زمانی نمایش داده می‌شود که پرداخت کاربر به هر دلیلی ناموفق باشد و پیام شکست پرداخت را نمایش می‌دهد.

**نکات مهاجرت به دات‌نت:**
- نمایش پیام شکست پرداخت.
- امکان تلاش مجدد یا بازگشت به سبد خرید.
