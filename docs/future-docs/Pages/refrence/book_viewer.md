# صفحه مشاهده کتاب (Book Viewer)

**مسیر احتمالی:** `/refrence/book/<book_id>/`

**کاربرد:**
این صفحه اطلاعات کامل یک کتاب را نمایش می‌دهد؛ شامل عنوان، نویسنده، توضیحات، تصویر جلد و لینک‌های مرتبط.

**نکات مهاجرت به دات‌نت:**
- نمایش اطلاعات کتاب به صورت داینامیک.
- رعایت سئو و ساختار مناسب برای موتورهای جستجو.
