# صفحه جزئیات نویسنده (Author Detail)

**مسیر احتمالی:** `/refrence/author/<author_id>/`

**کاربرد:**
این صفحه اطلاعات کامل یک نویسنده را نمایش می‌دهد؛ شامل نام، بیوگرافی، تصویر و لیست کتاب‌های مرتبط.

**نکات مهاجرت به دات‌نت:**
- نمایش اطلاعات نویسنده به صورت داینامیک.
- نمایش لیست کتاب‌های نویسنده.
